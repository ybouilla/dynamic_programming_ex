## Largest subsequence


Find among 2 strings of characters, the largest common subsequence

Examples:
```
 Input: s1 = “ABC”, s2 = “ACD”
 Output: 2

 Input: s1 = “AGGTAB”, s2 = “GXTXAYB”
 Output: 4

 Input: s1 = “ABC”, s2 = “CBA”
 Output: 1
 ```
